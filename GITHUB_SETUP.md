## Step 1: Initialize Git Repository

```bash
# Initialize git
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: Placement Assistance Copilot - AI-powered placement evaluation system"
```

## Step 2: Create GitHub Repository

1. Go to [GitHub](https://github.com/new)
2. Repository name: `Placement-Assistance-Copilot`
3. Description: (Use the description provided below)
4. Choose **Public** or **Private**
5. **DO NOT** initialize with README, .gitignore, or license (we already have them)
6. Click **Create repository**

## Step 3: Connect and Push

```bash
# Add remote (replace with your actual username if different)
git remote add origin https://github.com/Bhupendra-DS/Placement-Assistance-Copilot.git

# Rename branch to main (if needed)
git branch -M main

# Push to GitHub
git push -u origin main
```
